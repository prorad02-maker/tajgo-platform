# Map QA Checklist

## Клиент

- [ ] Карта открывается на Худжанде и мягко показывает GPS при разрешении.
- [ ] A выбирается через GPS, поиск и ручной пин.
- [ ] B выбирается через поиск и ручной пин.
- [ ] Дубли не подтверждаются автоматически.
- [ ] После B видны маршрут, distance, ETA и цена.
- [ ] «Показать весь маршрут» помещает A/B в кадр.
- [ ] Без сети local/recent/manual не падают.
- [ ] Голые координаты не являются главным адресом.

## Курьер

- [ ] Accepted ведёт к A, pickedUp — к B.
- [ ] Follow включён при открытии.
- [ ] Жест выключает follow, GPS-кнопка возвращает.
- [ ] Главная кнопка видна и соответствует статусу.
- [ ] Road route показывает «Маршрут построен».
- [ ] Direct fallback показывает «Маршрут предварительный».
- [ ] Off-route предупреждает мягко и не блокирует действие.
- [ ] Слабый GPS показывает понятный русский текст.

## Диспетчер

- [ ] «Показать весь город» возвращает обзор Худжанда.
- [ ] Фильтры Все/Курьеры/Ждут/В пути работают.
- [ ] Маршрут рисуется только у выбранного заказа.
- [ ] Peek-карточка закрывается и открывает детали.

Обязательный финальный прогон выполняется на физическом Android-телефоне.
