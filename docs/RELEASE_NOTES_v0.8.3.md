# TajGo v0.8.3 — Brand Assets + Splash Refresh

## Что изменилось

- Добавлен новый фирменный знак TajGo: маршрут, движение вперёд и горы Худжанда.
- Добавлены светлый и тёмный splash-фоны 1080×1920 с панорамой Худжанда.
- SplashScreen получил адаптивную композицию, центральный логотип, название TajGo, слоган «Проще. Быстрее. Честнее. Для своих.», подпись города и фирменный progress bar.
- Исправлен пользовательский текст ошибки подключения: техническое исключение больше не показывается на первом экране.
- Добавлены безопасные fallback-состояния, если графический ассет недоступен.
- Launcher-иконки Android/iOS сгенерированы из `assets/brand/app_icon.png`; для Android добавлены adaptive и monochrome слои.
- Версия приложения повышена до `0.8.3+11`.
- Android label сохранён как `TajGo`.

## Почему раньше оставался старый дизайн

Каталог `assets/brand/` содержал только `.gitkeep`. Splash пытался открыть отсутствующие PNG и переходил на временный кодовый логотип `T+`. Конфигурация `flutter_launcher_icons` уже ссылалась на `app_icon.png`, но генератор нельзя было запускать без реальных исходников, поэтому Android продолжал использовать стандартные старые mipmap-иконки.

## Не изменялось

- Firebase project и Firebase deploy.
- `applicationId` / namespace `tj.tajgo.app`.
- `firebase_options.dart`.
- Логика клиента, курьера, заказов, навигации и админки.
