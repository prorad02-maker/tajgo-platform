# TajGo Pilot Core — release notes

Внутренняя продуктовая веха `v0.9.0 Pilot Core` собрана поверх фактической линии приложения `1.3.0+28` (версия не понижалась с `1.2.0+27`).

- постоянный onboarding роли;
- компактная клиентская главная;
- курьерские вкладки по расстоянию;
- walking удалён с legacy fallback на bicycle;
- строгая граница `<1000 м`, свежий GPS и один активный заказ;
- цена клиента и транзакционные courier offers;
- выбор курьера клиентом;
- внешний навигатор с безопасным geo fallback;
- локально обновлены Rules/indexes, deploy не выполнялся;
- добавлено 15 offline acceptance-тестов.

Перед физическим пилотом нужны локальная проверка на двух аккаунтах, deploy одобренных Rules и проверка реальной точности GPS у границы радиуса.
