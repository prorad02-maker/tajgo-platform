# TajGo Map Core Architecture

## Слои

- `PlaceSearchService` объединяет recent, локальный gazetteer и remote geocoding.
- `AddressNormalizer` отвечает за сокращения и aliases.
- `ReverseGeocodingService` превращает выбранную точку в короткий адрес и всегда
  возвращает безопасный `Точка на карте` fallback.
- `RouteService` выбирает road-provider, cache или direct fallback.
- `TajGoMapCamera` выполняет отменяемые и throttled-анимации камеры.
- Экраны получают готовые модели и не знают формат ответа внешних API.

## Поиск

Remote-поиск запускается после 2 символов с debounce 320 мс. Пустой запрос не
идёт в сеть: UI показывает отдельно «Недавние», «Рядом с вами» и «Популярные».
Результат только перемещает карту; A/B фиксируются отдельной кнопкой.

Recent places хранят до 20 точек локально, сохраняют тип pickup/dropoff, usedAt,
удаляют дубли и игнорируют повреждённое хранилище.

## Карта и точки

A — зелёная точка забора, B — синяя точка доставки, пользователь — синяя GPS-
точка. После A автоматически активируется B. После B один раз строится маршрут,
показываются distance, ETA и цена. Технические ошибки provider клиенту не
показываются.
