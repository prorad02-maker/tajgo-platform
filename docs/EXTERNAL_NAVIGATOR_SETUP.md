# Внешний навигатор

В настройках курьера доступны TajGo, Яндекс Навигатор, Google Maps, 2GIS и системный навигатор. Выбор и `askEveryTime` сохраняются локально. Автоматическое открытие после принятия выключено.

- `accepted` открывает координаты A;
- `pickedUp` открывает координаты B;
- если deep link недоступен, используется `geo:lat,lng`.

Передаются только координаты цели. Код получения, телефон и приватный комментарий в URI не включаются. Встроенная навигация TajGo остаётся доступна всегда.
