# TajGo Foundation v2.1

Первый фирменный интерфейс TajGo+ для запуска на Android.

## Что есть

- Зелёная палитра под локальность, доверие и экологичную доставку.
- Логотип T+.
- Экран приветствия.
- Выбор ролей: клиент, курьер, админ.
- Первый экран курьера для Худжанда.
- Разделение кода на theme / screens / widgets.

## Что дальше

- Подключение Firebase.
- Экран регистрации по телефону.
- Анкета курьера с документами.
- Карта и заказ.
