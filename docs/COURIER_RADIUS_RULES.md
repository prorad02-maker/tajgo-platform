# Радиус принятия

Единственное правило пилота: `distanceMeters < 1000` от актуального положения курьера до pickup A.

- 999 м — доступно;
- 1000 м — недоступно;
- 1001 м — недоступно.

Перед предложением и повторно при выборе клиента проверяются: наличие GPS, возраст позиции не более 15 секунд, наличие accuracy, accuracy не хуже 50 м, отсутствие `activeOrderId` и `isBusy != true`.

При stale GPS показывается «Обновляем ваше местоположение…». При слабом GPS — спокойная подсказка дождаться точной позиции на открытом месте.
