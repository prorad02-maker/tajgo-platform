# TajGo v0.8.4 — Map Search & Address UX

## Добавлено

- единый поиск адресов для точек «Откуда» и «Куда»;
- локальный offline-справочник коротких названий Худжанда;
- нормализация сокращений и aliases;
- результаты с адресом, расстоянием, категорией и уровнем соответствия;
- предупреждение и ручной выбор при нескольких похожих местах;
- локальная история последних 10 подтверждённых мест;
- использование текущего GPS как предлагаемой точки забора;
- reverse-geocoding центра карты и понятный координатный fallback;
- разные маркеры A/B и явная кнопка подтверждения активной точки.

## Не изменено

Firebase, Phone Auth, `tj.tajgo.app`, GPS-механика курьера, расчёт цены,
создание заказа и курьерская навигация не менялись.

## Ограничения

Текущий удалённый поиск использует доступный системный geocoding provider и
является fallback для демо. Для production нужен выбранный поставщик геокодинга,
проверенные локальные координаты, кэш и контроль лимитов.
