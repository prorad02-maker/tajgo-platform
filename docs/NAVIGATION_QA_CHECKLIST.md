# Navigation Pro — чек-лист телефона

Основан на `MAP_NAVIGATION_MASTER_QA.md`.

## Курьер

- [ ] При открытии карта следует за курьером.
- [ ] Большая верхняя карточка показывает следующую цель, расстояние и ETA.
- [ ] При дорожном маршруте видна русская инструкция следующего манёвра.
- [ ] Ручное перемещение карты не борется с пальцем и включает free mode.
- [ ] GPS-кнопка возвращает follow mode.
- [ ] Кнопка маршрута показывает маршрут целиком.
- [ ] Слабый GPS даёт понятное предупреждение и не ломает кнопки.
- [ ] В радиусе 120 м появляется «Вы рядом с точкой».
- [ ] После «Забрал заказ» цель явно меняется с A на B.
- [ ] Комментарий/ориентир виден в карточке текущей точки.
- [ ] Off-route появляется только после устойчивого отклонения и исчезает после перестроения.

## Клиент

- [ ] До забора показывается ETA курьера до A.
- [ ] После забора показывается ETA до B.
- [ ] В радиусе 120 м появляется «Курьер рядом».
- [ ] На карте видны A, B, курьер и собственная позиция клиента.
- [ ] Кнопки «Моё местоположение» и «Показать маршрут» работают.
- [ ] Ошибка/timeout внешнего routing provider не показываются клиенту.

## Fallback и производительность

- [ ] Без `TAJGO_ROUTE_ENABLED` заказ полностью проходится по предварительному маршруту.
- [ ] Direct fallback не показывает ложный off-route.
- [ ] Админская карта рисует маршрут только выбранного заказа.
- [ ] Фильтры online/waiting/active и «Показать весь город» работают плавно.
- [ ] Создание заказа с поиском и A/B укладывается в 90 секунд.
- [ ] Нет зависания главной кнопки во время перестроения.
