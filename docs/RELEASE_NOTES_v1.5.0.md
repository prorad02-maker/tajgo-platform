# TajGo 1.5.0 — Marketplace & Admin Expansion

Сборка `1.5.0+30` готовит полный пилот витрины Худжанда без изменения основной
статусной машины заказов.

## Новое

- Главные категории: Еда, Продукты, Цветы.
- Локальная preview-витрина: 6 заведений и 18 товаров при пустой базе.
- Защита от оформления локальных preview-данных.
- Админская публикация тестового каталога одной transaction.
- Ручное редактирование карточки партнёра и его ассортимента.
- JSON import schema v1: validation, dry-run, предупреждения и audit log.
- Сортировка партнёров и товаров через `sortOrder`.
- Поиск адреса на marketplace-карте, debounce, текущая геопозиция и обзор
  маршрута.
- Клиентский tracking рисует отдельный live-маршрут курьера до текущей цели и
  показывает свежесть GPS.
- Исправлена совместимость OSRM: удалён неподдерживаемый `language=ru`.
- Android package visibility добавлена для Яндекс Навигатора, Google Maps,
  2GIS и системного `geo:`.
- Firestore Rules закрывают неактивных партнёров и скрытые товары от клиентов и
  валидируют публичную схему marketplace.

## Ограничения пилота

- Публичный OSRM не имеет production SLA; при ошибке остаётся честная прямая
  линия и fallback ETA.
- Магазины и изображения должен проверить владелец; локальные карточки явно
  являются примерами.
- Остатки и расчёт каталога защищены transaction + Rules, но для масштабного
  запуска рекомендована серверная Callable Function с App Check.
- Firestore Rules/Indexes не задеплоены автоматически.
